What Is the Role of Attention in Feature Search Versus Conjunction Search?

Searching for a book with a blue cover in a large bookcase would be quick and easy if all the other books had red covers. In laboratory studies of this type of visual search, a display containing multiple objects is presented on a computer screen, and the participant is instructed to search the display for a target with one or more specific features; the target is present half the time and absent the other half. Searching for a target that differs from all the other objects in the display in one feature is called feature search, as in the example of the book with the blue cover. Searching for a target that is defined by multiple features is called conjunction search, because the participant is searching for a target that consists of a conjunction of features in a single object.

In an experiment like the one in this demonstration (illustrated in Figure 1), where the target is the blue horizontal bar, the participant responds by pressing a button as soon as she finds the target or as soon as she determines that the target is absent. This procedure is repeated many times, with varying numbers of objects in the display, and the time required to make each response is recorded. The left and middle displays in the figure illustrate feature search—in the left display; the target is the only blue bar; in the middle display, the target is the only horizontal bar; and in both cases, the search is quick and easy because the single feature that defines the target “pops out” of the display. The graph in the figure shows that the response time in feature search is fast regardless of the number of objects in the display and regardless of whether the target is present or absent.

The right display illustrates conjunction search: the participant must search for the conjunction of features blue and horizontal, because the display contains other blue objects and other horizontal objects. The graph shows that conjunction search is much more difficult than feature search: the response time is greater than in feature search for any given number of objects, the response time increases as the number of objects increases, and the response time is greater when the target is absent than when the target is present.

These results suggest that, in a feature search, the observer can detect the unique feature by attending to the entire visual scene—the target “pops out.” In contrast, conjunction search requires the observer to direct attention to one object at a time, sequentially, to determine whether it contains the required conjunction of features. This distinction between rapid, efficient feature search and slow, inefficient conjunction search provides good evidence that binding features together requires attention to the spatial location of those features, to make sure that the features are in the same location and belong to the same object.
